# site-macaujck-lab

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。仓库内容不针对任何特定域名或具体网站，主要目的是保存和共享一些静态页面资源。

## 目录说明

```
site-macaujck-lab/
├── pages/          # 存放独立 HTML 页面文件
├── assets/         # 页面所需的静态资源（CSS、JS、图片等）
├── README.md       # 本文件
└── LICENSE         # 许可证文件（如有）
```

- `pages/`：每个子目录或文件代表一个独立的页面，命名清晰，便于识别。
- `assets/`：按页面或功能组织，避免资源混淆。

## 页面归档说明

- 所有页面均为纯前端实现，无后端依赖。
- 页面之间相互独立，不共享状态或数据。
- 部分页面可能包含演示性或实验性内容，请按需使用。
- 归档的页面仅作技术展示或参考用途，不保证其长期可用性。

## 维护说明

- 本仓库不定期更新，新增页面或修正现有内容。
- 欢迎通过 Issue 提交建议或报告问题。
- 不接受直接修改主分支的 Pull Request，如有改进建议，请先开 Issue 讨论。

## 使用方式

你可以直接通过 GitHub Pages 或本地打开 `pages/` 下的 HTML 文件来查看页面。如需部署，请自行配置静态文件服务。
